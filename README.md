| Datei | Inhalt |
|-------|--------|
| `index.html` | Landingpage |
| `privacy.html` | Datenschutzerklärung (DE) |
| `privacy-en.html` | Privacy Policy (EN) |

| `style.css` | Gemeinsames Stylesheet |

## Website aktualisieren

```bash
cd ~/development/projects/terminder-website

# Dateien bearbeiten, dann:
git add .
git commit -m "Beschreibung der Änderung"
git push
```

GitHub Pages deployt automatisch vom `main`-Branch. Die Änderungen sind nach ca. 1–2 Minuten live.

## Konfiguration

- **Repo:** https://github.com/noah-bl/terminder
