2. Datenerhebung

Terminder greift ausschließlich lesend auf deinen Google Kalender zu. Folgende Daten werden erhoben:

  • Google OAuth-Token: Wird im Rahmen von Google Sign-In abgerufen, um den Zugriff auf die Google Calendar API zu autorisieren.
  • Google User-ID: Wird im Rahmen von Google Sign-In erhoben, um dein Konto zu identifizieren.
  • IP-Adresse: Wird von Google Sign-In temporär zur Betrugsprävention erfasst. Terminder selbst erhebt oder speichert keine IP-Adressen.
  • Kalender- und Termindaten: Werden von der Google Calendar API abgerufen, einschließlich Kalendernamen, Terminbezeichnungen, Datum, Uhrzeit und Ort.
  • Benutzereinstellungen: Deine Alarm-Einstellungen wie Vorlaufzeit, ausgeschlossene Kalender, ausgeschlossene Termine und Pause-Modus-Konfiguration.

Terminder erhebt keine weiteren personenbezogenen Daten über die oben genannten hinaus. Es werden keine Analyse-, Werbe- oder Nutzungsdaten erfasst.

3. Datenverwendung

Terminder verwendet die erhobenen Google-Nutzerdaten ausschließlich für folgende Zwecke:

  • Google OAuth-Token: Wird ausschließlich zur Authentifizierung von API-Anfragen an die Google Calendar API in deinem Namen verwendet. Das Token wird an keinen anderen Dienst oder Server übermittelt.
  • Google User-ID: Wird ausschließlich verwendet, um deine lokalen App-Daten deinem Google-Konto zuzuordnen, damit Einstellungen nach erneuter Anmeldung wiederhergestellt werden können.
  • Kalender- und Termindaten: Werden ausschließlich verwendet, um deine bevorstehenden Kalendertermine zu lesen und lokale Geräte-Alarme basierend auf deinen konfigurierten Vorlaufzeiten zu planen. Die Verarbeitung erfolgt vollständig auf deinem Gerät.
  • Benutzereinstellungen: Werden ausschließlich zur Konfiguration des Alarm-Verhaltens gemäß deinen Präferenzen verwendet.

Terminder verwendet Google-Nutzerdaten nicht für Werbung, Analysen, Marktforschung, KI-/ML-Modelltraining oder andere Zwecke außer der Bereitstellung der Kern-Alarmfunktionalität der App. Deine Daten werden niemals verkauft, weitergegeben oder an Dritte übertragen.

4. Datenspeicherung

Alle Daten werden ausschließlich auf deinem Gerät gespeichert:

  • OAuth-Token: Sicher im iOS Keychain auf deinem Gerät gespeichert.
  • Kalenderdaten und Einstellungen: Im lokalen Speicher der App auf deinem Gerät gespeichert.

Terminder betreibt keine eigenen Server oder Cloud-Infrastruktur. Es werden keine Nutzerdaten an einen von uns kontrollierten Server übermittelt oder dort gespeichert. Die einzige Netzwerkkommunikation findet direkt zwischen deinem Gerät und den Google-Servern statt (Google Calendar API und Google Sign-In).

5. Weitergabe an Dritte

Terminder gibt keine Nutzerdaten an Dritte weiter, verkauft oder überträgt sie. Die einzige externe Kommunikation erfolgt mit:

  • Google Calendar API: Zum Abrufen deiner Kalender und Termine (nur Lesezugriff). Es gelten die Datenschutzbestimmungen von Google.
  • Google Sign-In: Zur Authentifizierung deines Google-Kontos. Es gelten die Datenschutzbestimmungen von Google.

6. Datenaufbewahrung

Deine Daten werden so lange auf deinem Gerät aufbewahrt, wie du die App nutzt:

  • Abmeldung (Logout): Das OAuth-Token wird sofort von deinem Gerät entfernt. Deine Einstellungen (Vorlaufzeiten, Kalender-Ausschlüsse etc.) bleiben lokal erhalten, damit sie bei erneuter Anmeldung wiederhergestellt werden können.
  • App-Löschung: Alle lokalen Daten – einschließlich Token, Kalenderdaten und Einstellungen – werden vollständig und dauerhaft von deinem Gerät entfernt.
  • Zugriff widerrufen: Du kannst den Zugriff von Terminder auf dein Google-Konto jederzeit unter myaccount.google.com/permissions widerrufen.

7. Deine Rechte (DSGVO)

Gemäß der Datenschutz-Grundverordnung (DSGVO) hast du das Recht auf:

  • Auskunft: Informationen über deine gespeicherten Daten anfordern (alle Daten befinden sich auf deinem Gerät und sind für dich zugänglich).
  • Löschung: Deine Daten löschen durch Abmeldung (entfernt das OAuth-Token) oder Deinstallation der App (entfernt alle Daten).
  • Widerruf: Den Zugriff von Terminder auf dein Google-Konto unter myaccount.google.com/permissions widerrufen.
  • Beschwerde: Eine Beschwerde bei einer Aufsichtsbehörde einreichen, wenn du der Meinung bist, dass deine Daten unrechtmäßig verarbeitet werden.

8. Analyse und Tracking

Terminder verwendet keine Analyse-Tools, Tracker, Cookies oder Werbe-SDKs. Es werden keine Nutzungs-, Verhaltens- oder Gerätedaten erfasst oder übermittelt.
